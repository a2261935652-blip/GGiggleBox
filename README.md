# GiggleBox Order Flow

这是一个基于 React + Tailwind CSS 的简易电商订单流程网站。

## 如何在本地运行

1.  确保你安装了 [Node.js](https://nodejs.org/).
2.  下载本项目所有文件。
3.  打开终端（Terminal），进入项目目录。
4.  安装依赖：
    ```bash
    npm install
    ```
5.  启动本地服务器：
    ```bash
    npm run dev
    ```
6.  在浏览器打开显示的地址（通常是 http://localhost:5173）。

## 如何发布给顾客（无需登录）

推荐使用 **Vercel** 或 **Netlify** 进行免费托管。

1.  将本项目上传到 GitHub。
2.  登录 [Vercel](https://vercel.com) 或 [Netlify](https://www.netlify.com)。
3.  导入你的 GitHub 项目。
4.  点击 Deploy。
5.  你会获得一个公开的链接（例如 `https://your-shop.vercel.app`），顾客点击该链接即可直接使用，无需登录。
